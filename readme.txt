== Description ==**Limit Comments Per Day** Limit Comments Per Day limits the number of comments users can make per day. The daily maximum limit is variable and may be specified by the administrator.


== Installation === IMPORTANT = 

Please backup your wordpress database before you install/uninstall/activate/deactivate/upgrade Limit Comments Per Day.

Manual Installation

1. Go to Dashboard -> Installed Plugins -> Search for "Limit Comments Per Day" and 

2. Upload the "Limit Comments Per Day" folder to the `/wp-content/plugins/` directory.

3. Activate the plugin through the 'Plugins' menu in WordPress. 

4. Visit the Settings tab on the Dashboard and enter "Limit Comments". 

5. Enter a value for "Number of comments per DAY".

6. Enter your user message in "Message when user reach the limit".
